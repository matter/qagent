# QAgent V3.2

V3.2 是 3.0 架构的剥离式迁移版本，目标是把旧系统中仍有价值的资产重新录入、导入或重实现到 3.0 架构下，然后删除旧运行时、旧入口和冗余数据，并重构策略模块的下单、仓位控制和执行意图表达。

- [3.2 架构迁移与策略模块重构需求及开发计划](./3.2-architecture-migration-and-strategy-refactor-plan.md)

## 范围边界

- 包含：3.0 架构全面迁移、旧资产重录/导入/归档策略、旧运行时剥离、策略模块重构、订单与仓位控制模型、agent/human 验收里程碑。
- 不包含：数据源质量升级。该方向已按产品决策从 backlog 中删除，不进入 V3.2。
