构建一个量化系统
先面对美股构建全流程，后续再接入A股
这个系统定位是低频量化交易研究系统，可以人用，也可以agent访问系统功能，也可以人用agent辅助进行研究
这个系统不对接券商，不对接高频交易
这个系统对接日频数据
这个系统专注量化策略研究，生成，回测，优化。
该系统为单用户和agent服务，无需用户系统

研究流程为以下几种情况：
1. 提出假设 - 编写因子逻辑 - 回测因子指标 - 优化因子逻辑 - 回测指标 - 确认因子 - 保存因子至因子库
2. 因子库选择因子 - 查看因子指标 - 圈选多个因子 - 整合为特征集 - 保存特征集
3. 选择特征集 - 选择学习目标 - 设置模型参数（lightgbm）- 设置训练参数 - 训练模型 - 查看结果 - 保存特征逻辑和模型文件（保证可用于预测）
4. 基于模板编写策略 - 可以引用已有因子的值和已有模型的预测分数 - 定义买卖信号 - 选择仓位管理算法 - 回测策略 - 保存策略
5. 选择保存策略 - 选择日期 - 选择股票集合 - 生成当日买卖信号
6. 股票数据浏览 - 可以灵活流程查看本地股票k线数据 - 副图有成交量和macd等常见指标
7. 股票数据管理 - 可以更新股票日线数据 - 可以选择几个股票构成股票集合即股票分组管理

因子编写和策略编写用python脚本实现
